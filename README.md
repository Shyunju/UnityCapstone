### 스토리
스트렌지 마을의 촌장으로 부터 마을에 벌어지는 이상한 일들에 대해 조사해 달라고 요청을 받은 김탐정.

그는 바로 스트렌지 마을에 찾아가고 도착하자 마자 마을 안쪽 보석이 붙은 촌장의 동상을 폭파되고 보석을 도난당하는 사건에 휘말리게 된다.

스트렌지 마을에서 일어나고 있는 일들에 대해 주민들과 대화를 나누며 사건을 해결해 보자.
